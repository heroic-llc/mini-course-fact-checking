# AI Fact-Checking and Verification - Course Guide

## Course Context
This is a mini course for writingIO focused on verifying AI-generated information.

## Video Script Guidelines
- Target length: 1-2 minutes per video (roughly 900 characters per minute)
- Use "A I" (with space) for proper text-to-speech pronunciation
- Keep language practical and clear
- Focus on actionable verification techniques

## Lesson Topics
1. Why Verification Matters - Importance of fact-checking AI
2. How AI Makes Mistakes - Understanding AI errors
3. Verifying Claims - Fact-checking techniques
4. Checking Sources - Source evaluation methods
5. Verification Habits - Building consistent practices
