# AI Fact-Checking and Verification

A mini course on verifying AI-generated information and building good verification habits.

## Course Overview

This course teaches why verification matters, how AI can make mistakes, techniques for verifying claims, how to check sources, and building habits for consistent fact-checking.

## Lessons

1. **Why Verification Matters** - The importance of checking AI output
2. **How AI Makes Mistakes** - Understanding AI limitations and errors
3. **Verifying Claims** - Techniques for fact-checking AI responses
4. **Checking Sources** - Evaluating and validating sources
5. **Verification Habits** - Building consistent fact-checking practices
